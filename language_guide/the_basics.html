基本类型

Swift是iOS和OS X开发使用的一门新语言. 尽管如此, Swift中还是有很多部分和使用C、Objective-C开发时的体验很相似.

Swift提供了所有C和Objective-C中重要的类型, 包括Int(整数), Double和Float(浮点数), Bool(布尔值), String(文本). Swift提供了两个强大的集合类型, Dictionary和Array, 具体请参见Collection Types.

跟C一样, Swift通过一个指定的名称使用变量来存储和代表某些值. Swift中大量运用了那些值不可变的变量, 称作常量. 常量比在C中更强大, 它在Swift中贯穿始终来保证代码更加安全, 更加简洁.

除了我们熟悉的类型, Swift引入了Objective-C中没有的高级类型, 这就包括元组(可以创建和传递一组值), 它可以作为一个复合值从函数中返回.

Swift还引入了可选类型用来处理未知值. 可选类型的意思是"要么存在一个值, 要么根本没有值". 可选类型和Objective-C中使用指针时的nil很相似, 但是它适用任何类型, 不仅仅是类. 可选类型比nil更加安全明了, 是许多Swift强大功能的核心.

可选类型表明Swift是一个强类型语言. Swift帮助你明确每个值的类型. 如果你的代码希望得到传入一个String类型, 类型安全机制会防止你误传入一个Int类型. 这可以帮助你在开发过程中尽可能的捕捉和修复错误.

常量与变量

常量和变量把特定类型的值(如数字10或字符串"hello")与一个名称(如maximumNumberOfLoginAttempts或welcomeMessage)联系在一起. 常量的值一旦被设定就不能更改, 然而变量在后面可以被设置为一个不同的值.

声明常量与变量

常量与变量在使用之前必须被声明. 使用let关键词声明常量, var关键词声明变量. 举个栗子, 下面是关于如何使用常量和变量来追踪用户的登录尝试次数的代码:

let maximumNumberOfLoginAttempts = 10
var currentLoginAttempt = 0

这段代码可以这样理解:

"声明一个常量叫做maximumNumberOfLoginAttemps, 赋值为10. 然后声明一个变量叫做currentLoginAttempt, 初始值为0."

在这个栗子中, 允许登录的最大尝试次数被声明为常量, 因为最大值永远不变. 当前尝试次数被声明为变量, 因为这个值在每次登录失败后会增加.

你可以在同一行上声明多个常量或变量, 以逗号分隔:

var x=0.0, y=0.0, z=0.0

如果代码中的值不会变化, 请使用let关键词声明为常量. 只有存储的值需要变化的时候才使用变量.

类型注解

在声明常量或变量的时候, 你可以添加一个类型注解来表明这个常量或变量能存储什么样的值. 语法格式是: 在变量或常量后面添加冒号, 然后空格, 然后是类型名称.

下面这个栗子为一个叫做welcomeMessage的变量进行类型注解, 表明这个变量只能存储String类型:

var welcomeMessage: String

冒号的意思是"xx类型的...", 所以上面的代码可以这么理解:

"声明一个String类型的变量叫welcomeMessage."

这句话中"String类型的"是指"能存储任何String类型的值".

welcomeMessage变量现在可以被设置为任何的字符串:

welcomeMessage = "Hello"

在实际中你很少需要写类型注解. 如果你在一个常量或变量被定义的时候设置初始值, Swift通常会自己判断那个常量或变量的类型, 具体请参见(Type Safety and Type Inference)[]. 在上述welcomeMessage栗子中, 没有设置任何初始值, 所以welcomeMessage变量的类型通过类型注解指定而不是通过初始值判定.

命名常量和变量

你可以用大部分你喜欢的字符来命名常量和变量, 包括Unicode字符:

let π = 3.14159
let 你好 = "你好世界"
let 🐶🐮 = "dogcow"
